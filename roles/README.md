
# Terraform IAM Roles Module

This module generates generic IAM roles as follows.

## Human roles

* Devops
* Readonly
* Superuser
* Support

## Machine roles

* Metrics

## Miscellany

Since this module is recommended to be applied across all accounts, it comes
with some other goodies:-

* Password defaults
